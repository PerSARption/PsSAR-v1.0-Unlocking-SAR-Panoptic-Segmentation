## Introduction
### Contributions
We propose the first public SAR panoptic segmentation dataset, named PsSAR-v1.0. Derived from HRSID, this dataset contains 5604       high resolution SAR images and their corresponding annotationsfor panoptic segmentation tasks. Based on PsSAR-v1.0, we constructs a bench-mark and conducts evaluations on SOTA panoptic segmen-tation models.

## Dataset
#### To download PsSAR-v1.0, , please refer to [PsSAR-v1.0](https://github.com/H-EmbodVis/NAUTILUS/tree/dataset//百度网盘).
### Directory Structure of the Dataset
PsSAR-v1.0 follows the **COCO panoptic segmentation standard**.

Expected directory structure:
```shell
├── ps_sarship
	├── train2017
	├── val2017
	└── annotations: 
		├── instances_train2017.json		
		├── instances_val2017.json			
		├── panoptic_train2017.json			
		├── panoptic_val2017.json			
		├── panoptic_val2017	
		└── panoptic_train2017
```
### The pipeline of annotation
PsSAR-v1.0 dataset is implemented with the Labelme tool. First, a polygon is constructed by taking the four corners of the image as key points to generate an initial mask for the sea area. Next, refined annotation is conducted on the land area to produce a land mask. The existing ship instance annotations from the HRSID dataset are directly adopted to form a set of ship masks. Finally, the land mask and all ship masks are subtracted from the initial sea mask to obtain the final sea mask.

<div align="center">
    <img src="./ann_pipe.png" width="100%"/>
</div>

The original JSON annotation files generated by Labelme are converted into a format compliant with the COCO panoptic segmentation annotation standard via the **labelme_to_cocoPan.py**.

## Train
### Dataset Preparation and Preprocessing
Ensure that the [PsSAR-v1.0](https://github.com/H-EmbodVis/NAUTILUS/tree/dataset) is fully downloaded and correctly organized as described in the previous section. Once prepared, you can proceed with the training instructions below.
1. Download [swin_large_patch4_window12_384_22k.pth](https://huggingface.co/microsoft/swin-large-patch4-window12-384).
3. Train PsSAR-v1.0 in accordance with the instructions detailed below.
```shell
conda activate pansar
cd mmdetection
# Remove useless parameters and rename reamin parameters
# Before training, please make sure to modify the path of the checkpoint and others hyper-parameters in the script
# Start training
python tools/train.py configs/panoptic_fpn/panoptic-fpn_r50_fpn_1x_ship_1.py
```
