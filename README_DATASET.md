Dataset Information
Dataset Name
PsSAR-v1.0: Unlocking SAR Panoptic Segmentation

Original Dataset Source
This dataset is derived from HRSID: A High-Resolution SAR Images Dataset for Ship Detection and Instance Segmentation (GPL-3.0 license).
Original dataset link: https://github.com/chaozhong2010/HRSID

Original dataset reference:
Shunjun Wei ; Xiangfeng Zeng ; Qizhe Qu ; Mou Wang ; Hao Su ; Jun Shi. HRSID: A High-Resolution SAR Images Dataset for Ship Detection and Instance Segmentation . IEEE Access

Modifications & Contributions
Based on the original HRSID dataset, We redefined and annotated the panoptic segmentation labels, which unifies the detection of countable "things" (ships) and the segmentation of uncountable "stuff" (background, sea, land, etc.) into a single task.
